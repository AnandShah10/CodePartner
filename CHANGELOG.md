# Change Log

All notable changes to the "codepartner" extension will be documented in this file.

Check [Keep a Changelog](http://keepachangelog.com/) for recommendations on how to structure this file.

## [0.0.1] - 2026-04-03

### Added

- Initial release of CodePartner.
- Multi-agent orchestration and planning.
- Artifact registry for code and documentation.
- Azure OpenAI and standard OpenAI provider support.
- Full web search and workspace context awareness.
- Beautiful, high-performance UI with glassmorphism design.

### Changed

- Improved model listing for both OpenAI and custom endpoints.
- Added support for fetching Azure deployment names from settings.
- Refined UI/UX for a more premium experience.
