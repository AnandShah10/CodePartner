# Change Log

All notable changes to the "CodePartner" extension will be documented in this file.

## [0.0.1] - 2026-04-23

### Added (Initial Public Release)
- **Premiere Release**: Modernized CodePartner with a high-fidelity glassmorphism UI.
- **Planning vs. Fast Modes**: Switch between deep-dive roadmap planning and quick chat responses.
- **Global Skill System**: Save and reuse complex instructions across all projects (stored in `~/.codepartner`).
- **Robust Search/Replace Edits**: Safe, non-destructive file modification tool.
- **Cross-Platform Browser Tools**: Automated research and screenshots on Windows, Mac, and Linux.
- **Smart Workspace Retrieval**: Semantic content-aware search for file context.
- **Plan Tab & Task Tracker**: Visual progress tracking for long-running implementation tasks.

### Changed
- Replaced workspace-local `.codepartner` with a global user directory for cross-project persistence.
- Upgraded the streaming parser for improved reliability and structured plan extraction.
- Enhanced the UI with collapsible "Thinking" blocks and a dedicated Skills management tab.
